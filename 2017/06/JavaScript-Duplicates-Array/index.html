JavaScript取出陣列重複/不重複值的方法

發表於 2017-06-22 | 分類於 CODE | 0 comments

今天早上群組有人問了這樣的問題：
如何檢視[11, 22, 33, 44, 44, 55, 55, 66]的陣列重複值。
經討論後出現了各種解法，覺得收穫很多紀錄下來。

以下敘述我將變數分別設定為：

  1. origin: 傳入要判斷的陣列，都是[1, 2, 'a', 3, 1, 'b', 'a']
  2. result: 不重複的回傳值
  3. repeat: 重複的回傳值

※以下順序為依據討論回答的順序決定，並非最佳解順序。
