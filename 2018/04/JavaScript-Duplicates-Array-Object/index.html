因為物件跟物件之間無法比較，所以想法是透過轉字串來比較，
第一步先使用new Set並透過map＆JSON.stringify把每一個陣列內的物件轉為字串，
再透過展開運算符[...]來把剛才Set的結果轉為陣列，
最後再用一次map把裡面的物件字串透過JSON.parse再轉回物件。
